# AIP - zadaci & beleške

Repo sadrži zadatke, rešenja i beleške koje pravim tokom spremanja AIP-a.

Zadaci su podeljeni po oblastima (npr. `matrice` ).

U svakoj oblasti se uglavnom nalaze dva dodatna direktorijuma:

## `lab`

Zadaci sa laboratorijskih vežbi.

Najčešće se unutra nalazi samo zadatak koji mi se pao za pripremu, jer sam previše lenj da  uradim sve. 

## `racunske_vezbe`

Sadrži:

* zadatke sa računskih vežbi,
* moja rešenja,
* beleške,
* neke dodatne gluposti
